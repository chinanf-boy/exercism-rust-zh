# 雨滴-Raindrops

把一个数字转化成和其因数有关的字符串

- 如果 3 是数字的一个因数, 输出"Pling".
- 如果 5 是数字的一个因数, 输出"Plang".
- 如果 7 是数字的一个因数, 输出"Plong".
- 如果数字既没有 3 , 也没有 5 和 7 作为数字的因数, 直接输出数字.

## 实例

- 28 的因数是 1, 2, 4, **7**, 14, 28.
  - 用雨滴的话来说, 是简单的"Plong".
- 30 的因数是 1, 2, **3**, **5**, 6, 10, 15, 30.
  - 用雨滴的话来说, 是"PlingPlang".
- 34 有四个因数:1, 2, 17, 34.
  - 用雨滴的话来说, 是"34".
